# OSRS Bingo

A Battle Royale Bingo application for Old School RuneScape.

## Features

- **Team Management** - Create teams and add OSRS player accounts
- **Bingo Board** - Visual grid showing team progress
- **WOM Integration** - Sync player data from Wise Old Man API
- **Proof System** - Submit and approve tile completions
- **Real-time Scoreboard** - Live team rankings

## Tech Stack

- **Frontend**: React, Vite, TailwindCSS
- **Backend**: Node.js, Express
- **Deployment**: GitHub Pages (frontend), Render (backend)

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (new terminal)
cd frontend && npm install && npm run dev
```

Open http://localhost:5173

## Environment Setup

Create `backend/.env` with required variables. See `.env.example` for template.
